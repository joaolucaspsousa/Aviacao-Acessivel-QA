# Aviação-Acessível-QA

Este repositório contém um validador de respostas de API, comparando com um arquivo CSV.


### 🖥️ Pre-requisitos
É necessário instalar as dependências `pandas` e `json` antes de executar a validação.
___

### 🌀Clone do repositório

Clone este repositório via HTTPS: `https://github.com/joaolucaspsousa/aviacao-acessivel.git`

Entre na pasta do projeto: `cd aviacao-acessivel`
